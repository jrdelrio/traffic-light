## Project Overview

Welcome to the React Traffic Light project! This project is a simple exercise that I completed during my studies while learning React in [4Geeks Academy](https://4geeks.com/). It serves as a basic demonstration of how React components work together to create a functional user interface.

## About the Project

The React Traffic Light is exactly what it sounds like‚Äîa digital traffic light sign that can change it's turned on light on the user's command. It's a minimalistic yet effective way to explore the fundamental concepts of React, specially the useState hook in a didactic way.
